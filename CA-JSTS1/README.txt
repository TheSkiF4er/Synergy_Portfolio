# TO-DO Sprint Board

Готовое React-приложение для практического задания: рабочий стол, активный спринт и панель администратора.

## Что реализовано
- 3 страницы: Главная, Активный спринт, Панель администратора
- Вкладки Product / Backlog
- Фильтрация задач по участникам команды
- Канбан-доска активного спринта
- Форма создания задач с переиспользуемыми полями
- Автогенерация ID задачи в формате `XX-1234`
- Проверка длины описания и комментариев
- Валидация доступного времени спринта
- Создание спринта с автоматическим расчетом длительности
- Добавление участников команды
- Сохранение данных в `localStorage`
- Простая визуализация статистики команды

## Запуск
```bash
npm install
npm start
```

## Стек
- React
- TypeScript
- React Router
- CSS

## Примечание
Проект оформлен в стиле Create React App и готов к запуску после установки зависимостей.
