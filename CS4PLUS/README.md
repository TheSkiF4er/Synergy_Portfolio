# Интерактивный калькулятор (улучшенная версия)

Проект — простой калькулятор на чистых HTML/CSS/JS, улучшенный по UX и сопровождаемости.

## Что улучшено

1) **Расширенные математические функции + обработка ошибок**
- добавлены операции: степень, остаток, √, ln, sin/cos
- единый механизм ошибок (например, деление на ноль, ln(A) при A ≤ 0, √A при A < 0)

2) **UX**
- подсветка ошибок в полях ввода + текст ошибки под полем
- аккуратный блок результата с состоянием ошибки
- история вычислений (сохранение в `localStorage`)
- кнопки: «Очистить поля» и «Очистить историю»

3) **Модульная архитектура JavaScript**
- ES Modules: логика разделена на модули `parser`, `operations`, `history`, `ui`, `app`
- чтобы добавить операцию — достаточно дополнить список в `js/operations.js`

4) **Комментарии, документация, тесты**
- комментарии и JSDoc для основных функций
- простые тесты на Node.js без зависимостей (см. `tests/runTests.js`)

5) **Адаптивность интерфейса**
- сетка перестраивается на мобильных устройствах, кнопки становятся в один столбец

## Структура

```
CS4/
  index.html
  style.css
  js/
    app.js
    ui.js
    parser.js
    operations.js
    history.js
  tests/
    runTests.js
  package.json
```

## Запуск

Откройте `index.html` в браузере.

## Тесты

Тесты рассчитаны на Node.js (без сторонних библиотек):

```bash
npm test
```

> Если `npm` недоступен, можно запустить напрямую: `node tests/runTests.js`.
