# Русский Стандарт — бриф для переноса в Figma

## Что внутри

Этот файл повторяет логику презентации и помогает быстро собрать итог в Figma.

## Фреймы

- Desktop: 1440 x 3600
- Tablet: 768 x 2200
- Mobile: 390 x 2200

## Сетка

- Desktop: 12 columns, margin 80, gutter 24
- Tablet: 8 columns, margin 32, gutter 20
- Mobile: 4 columns, margin 16, gutter 16

## Визуальный принцип

- Основа: editorial fintech minimalism
- Фон: теплый светлый
- Основной акцент: красный CTA
- Hero: графитовый блок с крупной типографикой
- Карточки: мягкие радиусы, светлые цветовые подложки
- Без мультяшных персонажей и без пластиковой 3D-стилистики
