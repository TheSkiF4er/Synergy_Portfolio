# CHANGEBODY — пакет сдачи

Состав:
- report_changebody.pdf — основной отчет по заданию
- research/ — референс-карты и выводы по ресерчу
- wireframes/ — мета-прототипы (low-fi)
- detailed/ — детализированные экраны
- flows/ — userflow из meta и detailed прототипов
- prototype/ — кликабельный офлайн-прототип (открыть index.html)

Примечание:
Кликабельный прототип сделан в HTML как офлайн-замена интерактивного прототипа Figma, чтобы архив открывался без внешних сервисов.
